# Portfolio - Cam Duc Binh

Website portfolio ca nhan mot trang, dung HTML/CSS/JavaScript thuan va co the publish truc tiep bang GitHub Pages.

## Cach chay local

Mo file `index.html` trong trinh duyet, hoac chay server tinh:

```bash
python -m http.server 8000
```

Sau do truy cap `http://localhost:8000`.

## Publish len GitHub Pages

1. Tao repository moi tren GitHub.
2. Upload cac file `index.html`, `styles.css`, `script.js`, `README.md`.
3. Vao `Settings` -> `Pages`.
4. Chon source la branch `main`, folder `/root`.
5. Luu lai va doi GitHub Pages tao link public.
